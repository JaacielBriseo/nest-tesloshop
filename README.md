# **Teslo Ecommerce API**

1. Clone repository

2. Install node modules

```
yarn install
```

3. Rename **.env.template** to **.env**

4. Fill env variables

5. Get up DB with docker

```
docker-compose up -d
```

6. Execute seed

```
/api/seed
```

7. Run db

```
yarn start:dev
```
